  • Building Blender as a Python Module: Windows Guide

  • Steps



  • Step 1:  Link to steps to build Blender from Source


  • Step 2:  Use your mouse to open up CmakeLists.txt found at ..\blender-git\blender and Ctrl+F to find and set 3 options:


  • WITH_PYTHON_INSTALL=OFF,  WITH_AUDASPACE=OFF,   WITH_PYTHON_MODULE=ON


  • Step 3: Open cmd.exe and cd over to ..\blender-git\blender. Then run "make bpy". The build will be inside ..\blender-git


  • Step 4: Now, cd over to ..\blender-git\*long_build_folder_name*\bin\Release and then run the following commands:


  • > copy bpy.pyd C:\Python37\Lib\site-packages\

  • > copy *.dll C:\Python37\Lib\site-packages\

  • > del C:\Python37\Lib\site-packages\python37.dll

  • > xcopy /E bin\2.81 C:\Python37\




  • Gotchas



  • 1  You cannot use Visual Studio Community for this,  only  VS Code


  • 2  Python version for Blender2.8+ is 3.7 and is assumed here to be a global install
